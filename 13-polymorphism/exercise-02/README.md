# Exercice 2 : Polymorphisme avec Collections

## Objectif

Utiliser le polymorphisme avec des tableaux et collections pour traiter différents types d'objets de manière uniforme.

## Instructions

1. Créez un tableau de type Animal
2. Stockez différents types d'animaux (Chien, Chat)
3. Parcourez le tableau et observez le polymorphisme

## À Faire

- Créer un tableau Animal[]
- Y placer un Chien, un Chat, et un Animal
- Parcourir le tableau et appeler faireDuBruit() sur chaque élément
- Observer que chaque objet exécute sa propre version de la méthode

## Notes

- Le polymorphisme permet de traiter différents types comme le même type de base
- Très utile pour les collections : on peut stocker différents types dans la même collection
- Chaque objet garde son comportement spécifique malgré la référence commune

## Résultat Attendu

```
Le chien aboie
Le chat miaule
L'animal fait du bruit
```
