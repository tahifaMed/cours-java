# Exercice 2 : Plusieurs Classes Concrètes

## Objectif

Créer plusieurs classes concrètes qui étendent la même classe abstraite.

## Instructions

1. Utilisez la classe abstraite Forme de l'exercice précédent
2. Créez une classe Rectangle qui étend Forme
3. Implémentez les méthodes abstraites dans Rectangle

## À Faire

- Créer une classe Rectangle qui étend Forme
- Ajouter des attributs largeur et hauteur
- Implémenter calculerAire() et calculerPerimetre() pour un rectangle
- Créer des objets Rectangle et Cercle et tester

## Notes

- Plusieurs classes peuvent étendre la même classe abstraite
- Chaque classe concrète implémente les méthodes abstraites selon sa propre logique
- C'est le polymorphisme en action

## Résultat Attendu

```
Cercle - Aire : 78.53981633974483, Périmètre : 31.41592653589793
Rectangle - Aire : 20.0, Périmètre : 18.0
```
