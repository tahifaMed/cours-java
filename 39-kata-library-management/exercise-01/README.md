# Exercice 1 : Création et Manipulation de Base des String

## Objectif

Apprendre à créer des String et utiliser les méthodes de base : length(), charAt(), et la concaténation.

## Instructions

1. Ouvrez le fichier `Exercise01.java`
2. Complétez les TODOs pour créer et manipuler des String
3. Affichez les résultats
4. Compilez et exécutez le programme

## À Faire

- Créez une String avec votre prénom
- Créez une String avec votre nom
- Affichez la longueur de chaque String
- Affichez le premier caractère de chaque String
- Concaténez le prénom et le nom avec un espace entre les deux
- Affichez la longueur de la String résultante

## Notes

- Utilisez `length()` pour obtenir la longueur d'une String
- Utilisez `charAt(0)` pour obtenir le premier caractère
- Utilisez l'opérateur `+` pour concaténer des String

## Compilation et Exécution

```bash
javac Exercise01.java
java Exercise01
```

## Résultat Attendu

Le programme doit afficher les informations sur les String créées (longueur, premier caractère, concaténation).
