# Exercice 2 : Affichage Multiple

## Objectif

Créer un programme qui affiche plusieurs messages sur différentes lignes.

## Instructions

1. Complétez le fichier `Exercise02.java`
2. Le programme doit afficher trois messages :
   - "Premier message"
   - "Deuxième message"
   - "Troisième message"

## À Faire

- Utilisez `System.out.println()` pour chaque message

## Résultat Attendu

```
Premier message
Deuxième message
Troisième message
```
