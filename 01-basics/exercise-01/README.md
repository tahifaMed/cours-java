# Exercice 1 : Votre Premier Programme Java

## Objectif

Créer votre premier programme Java qui affiche "Bonjour, Monde !" dans la console.

## Instructions

1. Ouvrez le fichier `Exercise01.java`
2. Complétez le code aux endroits marqués par `TODO`
3. Compilez et exécutez le programme
4. Vérifiez que la sortie correspond à `ExpectedOutput.txt`

## À Faire

- Complétez la déclaration de la classe
- Ajoutez la méthode main
- Ajoutez une instruction pour afficher "Bonjour, Monde !"

## Compilation et Exécution

```bash
javac Exercise01.java
java Exercise01
```

## Résultat Attendu

Le programme doit afficher :
```
Bonjour, Monde !
```
