# Exercice 1 : Observer Pattern - Système de News

## Objectif

Implémenter le pattern Observer pour un système de distribution de news.

## Résultat Attendu

```
Breaking: Important event!
CNN received: Breaking: Important event!
BBC received: Breaking: Important event!
Al Jazeera received: Breaking: Important event!

Technology: New innovation announced
CNN received: Technology: New innovation announced
BBC received: Technology: New innovation announced
Al Jazeera received: Technology: New innovation announced

Breaking: Exclusive update
CNN received: Breaking: Exclusive update
Al Jazeera received: Breaking: Exclusive update
```
