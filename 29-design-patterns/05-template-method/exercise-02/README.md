# Exercice 2 : Template Method - Génération de Rapports

## Objectif

Créer un template pour générer différents types de rapports avec la même structure.

## Résultat Attendu

```
=== RAPPORT HTML ===
<html><head><title>Rapport</title></head>
<body>
<h1>Données importantes</h1>
</body></html>

=== RAPPORT PDF ===
PDF Document
=============
Données importantes
=============
End of PDF
```
