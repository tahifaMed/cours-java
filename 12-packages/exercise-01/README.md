# Exercice 1 : Création et Utilisation d'un Package

## Objectif

Apprendre à créer et utiliser des packages pour organiser les classes Java.

## Instructions

1. Créez une structure de package
2. Placez des classes dans le package
3. Utilisez les imports pour accéder aux classes

## À Faire

- Créer une classe Utilitaires dans le package `com.entreprise.utils`
- Créer une classe principale dans le package par défaut qui utilise Utilitaires
- Utiliser `import` pour accéder à la classe du package

## Notes

- Déclaration de package : `package nom.du.package;` (première ligne du fichier)
- Les dossiers doivent correspondre à la structure du package
- Import : `import nom.du.package.Classe;`
- Le package par défaut : classes sans déclaration package

## Structure des Fichiers

```
com/entreprise/utils/Utilitaires.java
Exercise01.java
```
