# Exercice 3 : Parcours de Tableaux

## Objectif

Apprendre à parcourir les éléments d'un tableau avec différentes méthodes : boucle for classique et boucle for-each (enhanced for).

## Instructions

1. Complétez le fichier `Exercise03.java`
2. Parcourez les tableaux avec une boucle for classique
3. Parcourez les tableaux avec une boucle for-each
4. Calculez la somme et trouvez le maximum

## À Faire

- Parcourir un tableau avec une boucle for classique (avec index)
- Parcourir un tableau avec une boucle for-each (sans index)
- Calculer la somme de tous les éléments
- Trouver le maximum dans le tableau

## Notes

- Boucle for classique : `for (int i = 0; i < tableau.length; i++)`
- Boucle for-each : `for (int element : tableau)`
- La boucle for-each est plus simple mais ne donne pas accès à l'index
- Pour la recherche du maximum, initialisez avec le premier élément

## Résultat Attendu

```
Parcours avec for classique :
10 20 30 40 50
Parcours avec for-each :
10 20 30 40 50
Somme des éléments : 150
Maximum : 50
```
