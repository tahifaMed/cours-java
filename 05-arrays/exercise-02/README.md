# Exercice 2 : Accès et Modification d'Éléments

## Objectif

Apprendre à accéder aux éléments d'un tableau par leur index et à les modifier.

## Instructions

1. Complétez le fichier `Exercise02.java`
2. Accédez aux éléments du tableau par leur index
3. Modifiez les valeurs des éléments
4. Affichez les résultats

## À Faire

- Accéder au premier élément (index 0)
- Accéder au dernier élément (index length-1)
- Modifier des éléments du tableau
- Afficher le tableau avant et après modifications

## Notes

- Les indices commencent à 0
- Le dernier élément est à l'index `tableau.length - 1`
- Accès : `tableau[index]`
- Modification : `tableau[index] = nouvelleValeur`
- Attention : un index hors limites provoque une ArrayIndexOutOfBoundsException

## Résultat Attendu

```
Tableau initial : 10 20 30 40 50
Premier élément : 10
Dernier élément : 50
Élément à l'index 2 : 30
Après modification :
Tableau modifié : 10 20 99 40 50
```
