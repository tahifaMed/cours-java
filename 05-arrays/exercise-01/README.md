# Exercice 1 : Création et Initialisation de Tableaux

## Objectif

Apprendre à déclarer, créer et initialiser des tableaux en Java de différentes manières.

## Instructions

1. Ouvrez le fichier `Exercise01.java`
2. Créez des tableaux en utilisant différentes méthodes d'initialisation
3. Affichez les éléments des tableaux
4. Compilez et exécutez le programme

## À Faire

- Créer un tableau d'entiers de taille 5 avec `new int[5]`
- Créer un tableau de String initialisé avec des valeurs `{...}`
- Créer un tableau de doubles initialisé avec des valeurs
- Afficher la longueur des tableaux avec `length`
- Afficher les éléments des tableaux

## Notes

- Syntaxe : `int[] tableau = new int[taille];` (valeurs par défaut)
- Syntaxe : `int[] tableau = {valeur1, valeur2, ...};` (initialisation directe)
- La propriété `length` donne la taille du tableau
- Les indices commencent à 0

## Compilation et Exécution

```bash
javac Exercise01.java
java Exercise01
```

## Résultat Attendu

Le programme doit afficher les informations sur les tableaux créés.
