# Exercice 4 : Manipulation de String (replace, toUpperCase, trim)

## Objectif

Apprendre à manipuler des String en utilisant replace(), toUpperCase(), toLowerCase(), et trim().

## Instructions

1. Ouvrez le fichier `Exercise04.java`
2. Complétez les TODOs pour manipuler des String
3. Affichez les résultats
4. Compilez et exécutez le programme

## À Faire

- Convertir une String en majuscules
- Convertir une String en minuscules
- Remplacer des caractères dans une String
- Remplacer des sous-chaînes dans une String
- Supprimer les espaces en début et fin de String
- Combiner plusieurs manipulations

## Notes

- `toUpperCase()` et `toLowerCase()` créent une nouvelle String
- `replace()` peut remplacer un caractère ou une sous-chaîne
- `trim()` supprime les espaces en début et fin, pas au milieu
- Les String sont immuables : chaque méthode retourne une nouvelle String

## Compilation et Exécution

```bash
javac Exercise04.java
java Exercise04
```

## Résultat Attendu

Le programme doit afficher les résultats des manipulations de String.
