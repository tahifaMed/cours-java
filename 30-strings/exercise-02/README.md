# Exercice 2 : Méthodes de String (substring, indexOf)

## Objectif

Apprendre à utiliser les méthodes substring() et indexOf() pour extraire et rechercher dans des String.

## Instructions

1. Ouvrez le fichier `Exercise02.java`
2. Complétez les TODOs pour utiliser substring() et indexOf()
3. Affichez les résultats
4. Compilez et exécutez le programme

## À Faire

- Trouvez l'index d'un caractère dans une String
- Extrayez une sous-chaîne à partir d'un index
- Extrayez une sous-chaîne entre deux index
- Trouvez l'index d'une sous-chaîne
- Combinez indexOf() et substring() pour extraire une partie spécifique

## Notes

- `indexOf(char)` retourne l'index de la première occurrence du caractère
- `substring(start)` retourne la String à partir de l'index start jusqu'à la fin
- `substring(start, end)` retourne la String de l'index start (inclus) à l'index end (exclus)
- Les indices commencent à 0

## Compilation et Exécution

```bash
javac Exercise02.java
java Exercise02
```

## Résultat Attendu

Le programme doit afficher les résultats des opérations substring() et indexOf().
