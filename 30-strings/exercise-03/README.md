# Exercice 3 : Comparaison de String

## Objectif

Apprendre à comparer des String correctement en utilisant equals() et equalsIgnoreCase() au lieu de ==.

## Instructions

1. Ouvrez le fichier `Exercise03.java`
2. Complétez les TODOs pour comparer des String
3. Affichez les résultats
4. Compilez et exécutez le programme

## À Faire

- Comparez deux String avec equals()
- Comparez deux String avec equalsIgnoreCase()
- Comparez deux String avec == (pour voir la différence)
- Vérifiez si une String commence par une certaine sous-chaîne
- Vérifiez si une String se termine par une certaine sous-chaîne

## Notes

- **IMPORTANT** : Utilisez toujours `equals()` pour comparer le contenu, jamais `==`
- `==` compare les références, pas le contenu
- `equalsIgnoreCase()` ignore la différence entre majuscules et minuscules
- `startsWith()` et `endsWith()` retournent un boolean

## Compilation et Exécution

```bash
javac Exercise03.java
java Exercise03
```

## Résultat Attendu

Le programme doit afficher les résultats des comparaisons de String.
