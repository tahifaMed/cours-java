# Exercice 4 : Boucles while et do-while

## Objectif

Apprendre à utiliser les boucles while et do-while pour répéter des actions tant qu'une condition est vraie.

## Instructions

1. Complétez le fichier `Exercise04.java`
2. Utilisez while pour répéter une action tant qu'une condition est vraie
3. Utilisez do-while pour garantir l'exécution au moins une fois
4. Comprenez la différence entre while et do-while

## À Faire

- Créer une boucle while qui compte de 0 à 4
- Créer une boucle while qui calcule la puissance de 2 jusqu'à 1000
- Créer une boucle do-while qui demande une confirmation (simulée)
- Créer une boucle do-while qui s'exécute au moins une fois même si la condition est fausse

## Notes

- Syntaxe while : `while (condition) { ... }`
- Syntaxe do-while : `do { ... } while (condition);`
- La différence : do-while s'exécute au moins une fois, même si la condition est fausse initialement
- Attention aux boucles infinies : assurez-vous que la condition devient fausse à un moment donné
