# Exercice 1 : Structures Conditionnelles (if-else)

## Objectif

Apprendre à utiliser les structures conditionnelles if, if-else et if-else-if pour prendre des décisions dans votre programme.

## Instructions

1. Ouvrez le fichier `Exercise01.java`
2. Complétez les structures conditionnelles selon les consignes
3. Utilisez if-else pour prendre des décisions basées sur des conditions
4. Compilez et exécutez le programme

## À Faire

- Utilisez if pour vérifier si un nombre est positif
- Utilisez if-else pour déterminer si une personne est majeure ou mineure
- Utilisez if-else-if pour attribuer une note selon une valeur numérique

## Notes

- La syntaxe : `if (condition) { ... }`
- La syntaxe : `if (condition) { ... } else { ... }`
- La syntaxe : `if (condition1) { ... } else if (condition2) { ... } else { ... }`
- Les accolades {} sont nécessaires si le bloc contient plusieurs instructions

## Compilation et Exécution

```bash
javac Exercise01.java
java Exercise01
```

## Résultat Attendu

Le programme doit afficher les résultats des différentes conditions testées.
