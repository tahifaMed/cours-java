# Exercice 5 : break et continue

## Objectif

Apprendre à utiliser break et continue pour contrôler le flux d'exécution dans les boucles.

## Instructions

1. Complétez le fichier `Exercise05.java`
2. Utilisez break pour sortir prématurément d'une boucle
3. Utilisez continue pour passer à l'itération suivante
4. Comprenez quand utiliser chaque instruction

## À Faire

- Utiliser break pour sortir d'une boucle quand une condition est remplie
- Utiliser continue pour ignorer certaines itérations
- Combiner break et continue dans des exemples pratiques

## Notes

- `break` : Sort immédiatement de la boucle (et du switch)
- `continue` : Passe à l'itération suivante de la boucle, en ignorant le reste du code de l'itération courante
- `break` sort complètement, `continue` passe juste à la suite
