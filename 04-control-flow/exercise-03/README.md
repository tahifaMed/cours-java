# Exercice 3 : Boucle for

## Objectif

Apprendre à utiliser la boucle for pour répéter des actions un nombre déterminé de fois.

## Instructions

1. Complétez le fichier `Exercise03.java`
2. Utilisez différentes variantes de la boucle for
3. Comprenez la syntaxe : `for (initialisation; condition; incrémentation)`

## À Faire

- Créer une boucle for simple qui affiche les nombres de 0 à 4
- Créer une boucle for avec un pas différent (par exemple, compter de 0 à 10 par pas de 2)
- Créer une boucle for décroissante (de 10 à 1)
- Calculer la somme des nombres de 1 à 10

## Notes

- Syntaxe : `for (int i = 0; i < 5; i++) { ... }`
- La partie d'initialisation s'exécute une seule fois au début
- La condition est vérifiée avant chaque itération
- L'incrémentation s'exécute après chaque itération
- La variable de boucle peut être déclarée dans la boucle (portée limitée au bloc)
