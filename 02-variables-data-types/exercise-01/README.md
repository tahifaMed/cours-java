# Exercice 1 : Déclaration et Initialisation de Variables

## Objectif

Apprendre à déclarer et initialiser différents types de variables en Java.

## Instructions

1. Ouvrez le fichier `Exercise01.java`
2. Déclarez et initialisez les variables demandées selon les types appropriés
3. Compilez et exécutez le programme
4. Vérifiez que la sortie correspond à `ExpectedOutput.txt`

## À Faire

- Déclarez une variable `age` de type `int` et initialisez-la à 25
- Déclarez une variable `prix` de type `double` et initialisez-la à 19.99
- Déclarez une variable `estActif` de type `boolean` et initialisez-la à `true`
- Déclarez une variable `lettre` de type `char` et initialisez-la à 'A'
- Affichez toutes ces variables

## Compilation et Exécution

```bash
javac Exercise01.java
java Exercise01
```

## Résultat Attendu

Le programme doit afficher les valeurs des variables dans un format clair.
