# Exercice 3 : Manipulation de Chaînes de Caractères

## Objectif

Apprendre à manipuler les objets String en Java : concaténation, méthodes courantes, et opérations de base.

## Instructions

1. Complétez le fichier `Exercise03.java`
2. Utilisez les méthodes de la classe String pour manipuler les chaînes
3. Effectuez les opérations demandées sur les chaînes de caractères

## À Faire

- Concaténez deux chaînes (prenom et nom) pour former un nom complet
- Utilisez la méthode `length()` pour obtenir la longueur d'une chaîne
- Utilisez `toUpperCase()` pour convertir en majuscules
- Utilisez `toLowerCase()` pour convertir en minuscules
- Utilisez `substring()` pour extraire une partie de la chaîne
- Utilisez `charAt()` pour obtenir un caractère à une position donnée

## Notes

- Les chaînes sont des objets immuables en Java
- L'opérateur `+` permet la concaténation
- Les indices commencent à 0

## Résultat Attendu

```
Nom complet : Jean Dupont
Longueur : 11
En majuscules : JEAN DUPONT
En minuscules : jean dupont
Sous-chaîne (0-4) : Jean
Caractère à l'index 5 : D
```
