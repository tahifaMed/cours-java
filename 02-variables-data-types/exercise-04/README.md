# Exercice 4 : Utilisation des Constantes

## Objectif

Apprendre à utiliser le mot-clé `final` pour créer des constantes et comprendre leur importance.

## Instructions

1. Complétez le fichier `Exercise04.java`
2. Déclarez des constantes avec le mot-clé `final`
3. Utilisez ces constantes dans votre programme
4. Essayez de comprendre pourquoi on ne peut pas modifier une constante

## À Faire

- Déclarez une constante `PI` de type `double` avec la valeur 3.14159
- Déclarez une constante `TAILLE_MAX` de type `int` avec la valeur 100
- Déclarez une constante `NOM_APPLICATION` de type `String` avec la valeur "Mon Application"
- Utilisez ces constantes dans des calculs ou affichages
- Commentez la ligne qui tente de modifier une constante (elle provoquerait une erreur)

## Notes

- Les constantes sont déclarées avec le mot-clé `final`
- Par convention, les constantes sont nommées en MAJUSCULES avec des underscores
- Une constante ne peut être initialisée qu'une seule fois
- Tenter de modifier une constante provoque une erreur de compilation

## Résultat Attendu

```
Nom de l'application : Mon Application
Valeur de PI : 3.14159
Taille maximale : 100
Aire d'un cercle de rayon 5 : 78.53975
```
