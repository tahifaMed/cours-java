# Exercice 1 : Créer un Enum Simple

## Objectif

Apprendre à créer et utiliser un enum de base pour représenter un ensemble fixe de constantes.

## Instructions

1. Créez un enum JourSemaine avec les jours de la semaine
2. Utilisez l'enum dans un programme
3. Parcourez toutes les valeurs de l'enum

## À Faire

- Créer enum JourSemaine avec valeurs : LUNDI, MARDI, MERCREDI, JEUDI, VENDREDI, SAMEDI, DIMANCHE
- Créer une variable de type JourSemaine
- Afficher la valeur de l'enum
- Parcourir toutes les valeurs avec values()
- Afficher chaque jour

## Notes

- Enum = ensemble fixe de constantes nommées
- Plus sûr et plus expressif que des constantes numériques
- Les valeurs sont automatiquement publiques, statiques et finales
- Méthode values() retourne un tableau de toutes les valeurs

## Résultat Attendu

```
Aujourd'hui : LUNDI
Tous les jours de la semaine :
LUNDI
MARDI
MERCREDI
JEUDI
VENDREDI
SAMEDI
DIMANCHE
```
