# Exercice 3 : Utilisation des Arguments

## Objectif

Créer un programme qui lit et affiche les arguments passés en ligne de commande.

## Instructions

1. Complétez le fichier `Exercise03.java`
2. Le programme doit :
   - Afficher le nombre d'arguments reçus
   - Afficher chaque argument sur une ligne séparée

## À Faire

- Utilisez `args.length` pour obtenir le nombre d'arguments
- Utilisez une boucle pour afficher chaque argument
- Utilisez `args[i]` pour accéder à l'argument à l'index i

## Exécution avec Arguments

```bash
javac Exercise03.java
java Exercise03 un deux trois
```

## Résultat Attendu (avec les arguments ci-dessus)

```
Nombre d'arguments : 3
Argument 0 : un
Argument 1 : deux
Argument 2 : trois
```
