# Exercice 5 : StringBuilder

## Objectif

Apprendre à utiliser StringBuilder pour construire et modifier efficacement des chaînes de caractères.

## Instructions

1. Ouvrez le fichier `Exercise05.java`
2. Complétez les TODOs pour utiliser StringBuilder
3. Affichez les résultats
4. Compilez et exécutez le programme

## À Faire

- Créer un StringBuilder
- Ajouter des String avec append()
- Insérer des String avec insert()
- Remplacer des parties avec replace()
- Supprimer des caractères avec delete()
- Convertir en String avec toString()

## Notes

- StringBuilder est mutable (contrairement à String)
- Utilisez StringBuilder pour de nombreuses modifications
- `append()` ajoute à la fin
- `insert()` insère à un index spécifique
- `replace()` remplace une portion entre deux index
- `delete()` supprime une portion entre deux index
- N'oubliez pas d'utiliser `toString()` pour obtenir une String

## Compilation et Exécution

```bash
javac Exercise05.java
java Exercise05
```

## Résultat Attendu

Le programme doit afficher les résultats des opérations StringBuilder.
