# Exercice 5 : Méthodes avec Paramètres et Valeurs de Retour

## Objectif

Créer une classe avec plusieurs méthodes qui acceptent des paramètres et retournent des valeurs.

## Instructions

1. Complétez la classe Calculatrice
2. Créez des méthodes avec différents types de paramètres et valeurs de retour
3. Testez toutes les méthodes dans Exercise05

## À Faire

- Créer une méthode `additionner(int a, int b)` qui retourne la somme
- Créer une méthode `multiplier(double x, double y)` qui retourne le produit
- Créer une méthode `estPair(int nombre)` qui retourne un boolean
- Créer une méthode `afficherMessage(String message)` qui ne retourne rien (void)

## Notes

- Les méthodes peuvent avoir zéro, un ou plusieurs paramètres
- Les méthodes peuvent retourner un type ou void (rien)
- Le type de retour est spécifié avant le nom de la méthode
- Utilisez `return` pour retourner une valeur
- Les méthodes void n'ont pas de return (ou `return;` sans valeur)

## Résultat Attendu

```
Somme : 8
Produit : 10.0
7 est pair ? false
Message : Bonjour !
```
