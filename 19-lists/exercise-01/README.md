# Exercice 1 : Utiliser ArrayList pour Stocker des Éléments

## Objectif

Apprendre à utiliser ArrayList pour créer et manipuler une liste d'éléments.

## Instructions

1. Créez une ArrayList pour stocker des chaînes de caractères
2. Ajoutez des éléments à la liste
3. Accédez aux éléments par index
4. Parcourez la liste
5. Utilisez les méthodes de base (size, contains, remove)

## À Faire

- Créer une ArrayList<String> pour stocker des noms
- Ajouter plusieurs noms avec add()
- Afficher la taille avec size()
- Accéder à un élément par index avec get()
- Vérifier si un élément existe avec contains()
- Parcourir la liste avec une boucle for-each
- Supprimer un élément avec remove()

## Notes

- ArrayList = tableau dynamique qui grandit automatiquement
- Permet les doublons et conserve l'ordre d'insertion
- Accès rapide par index (get, set)
- Import nécessaire : import java.util.ArrayList; et import java.util.List;

## Résultat Attendu

```
Taille de la liste : 4
Élément à l'index 0 : Alice
La liste contient 'Charlie' : true
Parcours de la liste :
Alice
Bob
Charlie
Diana
Taille après suppression : 3
```
