# Exercice 2 : Opérateurs d'Assignation

## Objectif

Apprendre à utiliser les opérateurs d'assignation combinés : +=, -=, *=, /=, %= qui permettent de modifier une variable de manière concise.

## Instructions

1. Complétez le fichier `Exercise02.java`
2. Utilisez les opérateurs d'assignation combinés pour modifier les variables
3. Affichez la valeur de la variable après chaque opération

## À Faire

- Initialisez une variable `x` à 10
- Utilisez `+=` pour ajouter 5 à x
- Utilisez `-=` pour soustraire 3 à x
- Utilisez `*=` pour multiplier x par 2
- Utilisez `/=` pour diviser x par 4
- Utilisez `%=` pour calculer x modulo 3

## Notes

- `x += 5` est équivalent à `x = x + 5`
- Ces opérateurs modifient directement la variable
- Affichez la valeur après chaque opération pour suivre les changements

## Résultat Attendu

```
x initial : 10
Après += 5 : 15
Après -= 3 : 12
Après *= 2 : 24
Après /= 4 : 6
Après %= 3 : 0
```
