# Exercice 6 : Opérateur Ternaire

## Objectif

Apprendre à utiliser l'opérateur ternaire (? :) qui permet de choisir une valeur selon une condition de manière concise.

## Instructions

1. Complétez le fichier `Exercise06.java`
2. Utilisez l'opérateur ternaire pour remplacer des instructions if-else simples
3. Affichez les résultats

## À Faire

- Utilisez l'opérateur ternaire pour déterminer si une personne est majeure
- Utilisez l'opérateur ternaire pour trouver le maximum entre deux nombres
- Utilisez l'opérateur ternaire pour déterminer si un nombre est pair ou impair

## Notes

- Syntaxe : `condition ? valeur_si_vrai : valeur_si_faux`
- C'est une alternative concise à if-else pour des choix simples
- Peut être imbriqué mais devient difficile à lire

## Résultat Attendu

```
Âge : 20
Statut : Majeur
Nombre 1 : 15
Nombre 2 : 25
Maximum : 25
Nombre : 8
Parité : Pair
```
