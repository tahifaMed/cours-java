# Exercice 1 : Opérateurs Arithmétiques

## Objectif

Apprendre à utiliser les opérateurs arithmétiques de base : addition (+), soustraction (-), multiplication (*), division (/), et modulo (%).

## Instructions

1. Ouvrez le fichier `Exercise01.java`
2. Effectuez les calculs demandés en utilisant les opérateurs arithmétiques
3. Affichez les résultats
4. Compilez et exécutez le programme

## À Faire

- Calculez la somme de deux nombres (10 + 5)
- Calculez la différence de deux nombres (10 - 5)
- Calculez le produit de deux nombres (10 * 5)
- Calculez le quotient de deux nombres entiers (10 / 3)
- Calculez le quotient de deux nombres décimaux (10.0 / 3.0)
- Calculez le reste de la division (modulo) : 10 % 3

## Notes

- La division entre entiers donne un résultat entier (troncature)
- La division avec des décimaux donne un résultat décimal
- Le modulo (%) retourne le reste de la division

## Compilation et Exécution

```bash
javac Exercise01.java
java Exercise01
```

## Résultat Attendu

Le programme doit afficher tous les résultats des calculs.
