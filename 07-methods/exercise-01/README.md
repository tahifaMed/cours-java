# Exercice 1 : Méthodes avec Paramètres et Valeurs de Retour

## Objectif

Apprendre à créer des méthodes qui acceptent des paramètres et retournent des valeurs de différents types.

## Instructions

1. Complétez la classe Utilitaires avec différentes méthodes
2. Créez des méthodes avec et sans valeur de retour
3. Testez toutes les méthodes dans Exercise01

## À Faire

- Créer une méthode `saluer()` sans paramètres ni retour (void)
- Créer une méthode `saluerPersonne(String nom)` avec un paramètre
- Créer une méthode `calculerCarre(int nombre)` qui retourne un int
- Créer une méthode `calculerMoyenne(double a, double b)` qui retourne un double

## Notes

- Syntaxe : `modificateur typeRetour nomMethode(paramètres) { ... }`
- `void` signifie qu'il n'y a pas de valeur de retour
- Utilisez `return valeur;` pour retourner une valeur
- Les paramètres permettent de passer des données à la méthode
- Une méthode peut avoir zéro, un ou plusieurs paramètres

## Résultat Attendu

```
Bonjour !
Bonjour Alice !
5 au carré = 25
Moyenne de 10.5 et 20.3 = 15.4
```
